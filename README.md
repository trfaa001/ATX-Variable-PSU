# ATX-Variable-PSU
DIY variable power supply using an ATX PSU and buck converter.

![Project Image](docs/photos/finished.jpg)

## Features
---

* Voltage: 0-50V
* Amperage: 8A
* Repurposing ATX PSU
* Low-cost
